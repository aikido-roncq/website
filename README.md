# Aïkido Roncq Website 🌐

Website of the Roncq's aïkido club.

## Prerequisites 📚

- [Node.js](https://nodejs.org/en/)
- [yarn](https://yarnpkg.com/getting-started/install)

## Run the project ▶ 

### Install dependencies

```bash
yarn
```

### Start the project

```bash
yarn dev
```

The website will be available on `http://localhost:3000`.

## Run tests 🧪

```bash
yarn test
```
